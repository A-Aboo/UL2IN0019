## About

This repository contains my SQL work from **TME 4 to TME 10** at **Sorbonne Université**.

The exercises were completed using **SQL** and the **H2 database**.  
The goal of this work is to understand relational databases step by step, from schema creation to advanced queries, updates, constraints, and SQL programming.
